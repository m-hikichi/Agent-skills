# Presentation Structures

## General Presentation

1. Title
2. Agenda
3. Context or problem
4. Main content
5. Summary
6. Q&A

## Project Update

1. Title
2. Executive summary
3. Progress
4. Outcomes
5. Risks
6. Next steps
7. Decision requests

## Tech Talk Or Lightning Talk

1. Title
2. Trigger or problem
3. Approach
4. Result or demo
5. Takeaways

## Guidance

- Add 1 or 2 setup slides when the audience needs context
- Remove basic explanation when the audience already knows the domain
- Split content before cutting important information
